Demo Section 2

All UI elements including:

  • Next button
  • Previous button
  • Done button
  • Skip button
  • "Don't show again" checkbox
  • Step counter (e.g., "2 of 4")

Are automatically localized!

The language system is flexible and easy to extend. You can add new languages by creating new files in the src/i18n/ directory.
